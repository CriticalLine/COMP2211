Frequently Asked Questions

Frequently Asked Questions

We are waiting for your questions! Please post your questions on Piazza, and we will update this section with common questions and answers.

Q: What if I get different words order in Task 2.2 and Task 2.3 when there's tie?
A: It doesn't matter your ZINC passing if you get all correct scores. We won't compare the order of the ties on ZINC. So just make sure you get the correct scores for all words and see if you can pass ZINC test.

Q: What is the output type of Task0.1?
A: There is a mistake in the notebook instruction. You should return a Pandas Series instead of a DataFrame. The return type should be "<'class' pandas.core.series.Series>". We have fixed this mistake in the notebook now and you will see it by redownloading it.

Q: What are the output types of Task 2.2 and Task 2.3?
A: You should return numpy arrays instead of lists.

Q: Can I use ".toarray()" in Task 1.2 and Task 1.3?
A: Yes you can use any numpy operation in this assignment. Only for-loops are not allowed to use.

Q: I do not use any for-loop in Task 3.1 but I still can not pass the efficiency test. How can I solve it?
A: Try to use summation instead of multiplication.

Note: for Task 1.1, 1.2 and 1.3, you should be able to pass the efficiency test if you only used numpy operations. If you fail in passing the efficiency tests, please email us or raise a question on Piazza.
