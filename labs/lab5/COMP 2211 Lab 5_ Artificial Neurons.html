Introduction

In this lab, we will use two datasets to demonstrate the use of the Artificial Neurons.

The first dataset we use is the Sonar dataset from the Machine Learning Repository of UCI. We will get the response signals from the sonar detectors of different objects.

https://archive.ics.uci.edu/ml/datasets/Connectionist+Bench+(Sonar,+Mines+vs.+Rocks)

Classifying underwater objects can be difficult for humans as our senses (like seeing, touching, and hearing) are restricted. Thus utilizing tools to help us conduct that is important. Therefore, we will classify the underwater objects as "rock" and "metal" using Artificial Neurons.

Apart from the Sonar dataset, we will also use the Iris dataset from the Machine Learning Repository of UCI. We will get the patterns for different iris plants.

https://archive.ics.uci.edu/ml/datasets/iris

We will classify the iris plants into "Iris Setosa" and "Iris Versicolor" using the Artificial Neurons.

End of Introduction
