End of Review

Introduction

Deep Learning algorithms incorporating neural networks are popular for various classification tasks, such as image classification, sentiment analysis, and classifying data into different labels. However, for this lab, you will be creating your Multilayer Perceptron (MLP) model for a regression problem: predicting students' final grades.



Dataset : UCI Student Performance

This is a popular dataset which serves as a good introduction to implementing regression tasks using machine learning and deep learning algorithms. The dataset consists of 649 samples, each described by 23 features and 1 target label (final grade G3).
Your task is to explore this dataset further, apply the necessary preprocessing techniques, and finally train a custom MLP model of your choice to predict final grades based on the 23 attributes.
End of Introduction
