Self-test 6:
Artificial Neural Network - Perceptrons

  1. Which of the following statements is/are true about a perceptron?

    1. It can only be used for linearly seperable data.
    2. It should have exactly two inputs and one output.
    3. One similarity between perceptron and KNN is that they both include learnable parameters.
    4. Every parameter must be updated during an epoch.
    Solution

    A

    B: It can have more than 2 inputs.

    C: There are no learnable parameters in KNN.

    D: Not necessarily all parameters are updated in one epoch.

  2. Which of the followings is/are true about a perceptron?

    1. It belongs to an unsupervised learning algorithm.
    2. It can be mathematically modeled as a vector dot product.
    3. Its output is equivalent to the direct result of a weighted sum of the inputs plus a bias.
    4. For a perceptron with n inputs and 1 output, there should be n+1 parameters, including bias.
    Solution

    B and D

    A: It is a supervised learning algorithm.

    C: The sum result needs to go through an activation function to obtain the final output.

  3. Suppose we have a dataset with many data points, each with N features. Each feature is either positive (with value 1) or negative (with value -1). We say a data point is "positive" if it has more positive features than negative features, and "negative" otherwise. Design the weights, bias and activation function of a perceptron to classify positive data points from negative data points.

    Solution

    A data point is positive if and only if the sum of all its values is strictly greater than 0. We can achieve this in a perceptron using the following settings:

    - Weights: all 1s.
    - Bias: 0
    - Activation function: Return 1 if the input is strictly greater than 0, and return 0 otherwise.

    When a data point is fed into the perceptron, we perform dot product on the data point with the weights vector. This sums all the values in the data point. Then, we check if this sum is strictly greater than 0 in the activation function. A data point is positive if and only if the activation function returns 1.

  4. Which of the following can be considered as hyperparameters for a Perceptron?

    1. choice of activation function
    2. choice of learning rate
    3. choice of having a bias term or not
    4. choice of the number of inputs
    Solution

    A, B and C

  5. Suppose we use perceptron to learn on K training samples where each sample is represented in n-dimensional space. We stopped training after e epochs. How many arithmetic operations were done during training? Express it in terms of K, e and n. (arithmetic operations include both multiplication and addition, ignore the operations for parameter update and the operations in the activation function)

    Solution

    Each sample will lead to n multiplications and n additions (don't forget the bias). That is 2n operations per sample. So total number of arithmetic operations during training is 2eKn.

  6. You are given a perceptron with 2 inputs, and weight w = [1, 1] and bias = 2; the chosen activation function is the step function, where it returns 1 when x > 0 and 0 otherwise.

    2. w = [100, 100], bias = 200

    ABD
