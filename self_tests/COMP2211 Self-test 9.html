  1. Which of the followings is/are true about the convolutional layer in CNN?

    1. The number of weights depends on the depth of the input volume.
    2. The number of biases is equal to the number of filters.
    3. The total number of parameters depends on the stride.
    4. The total number of parameters depends on the padding.
    Solution
    AB

  2. Which of the following statements claims certain advantages of CONV layers compared to Dense layers in computer vision tasks is/are TRUE?

    1. Convolution itself is a non-linear operation and brings more non-linearity and stronger expressiveness to the network.
    2. Conv layers usually have more parameters compared to Dense layers assuming the same input; hence it has stronger expressiveness.
    3. Convolution preserves spatial context while dense layers will lose spatial context by flattening everything.
    4. Conv layers are more similar to how the human visual system works, where different areas of the brain are activated by different levels of features.

  3. Suppose we have a conventional CNN architecture: input 32*32*3 RGB image, first Conv layer has 256 3*3 filters, followed by a 2*2 max pooling layer, followed by Conv layer with 128 3*3 filters, followed by another 2*2 max pooling layer, followed by Conv layer with 64 3*3 filters, then a 2*2 max pooling layer, then a flattening layer, a dense layer with 1000 neurons, and output layer with 10 neurons. What is the total number of parameters in the network, including bias, assuming no padding is done for any Conv layer?

    Solution

    3*3*3*256+256 + 3*3*256*128+128 + 3*3*128*64+64 + 2*2*64*1000+1000 + 1000*10+10 = 643010

    Hint: you need the output size after all the CONV layers to calculate the number of neurons after the Flatten layer.

  4. Read the following code in python, which builds a CNN using the Keras library. What is the number of parameters in this network?

    from keras.models import Sequential
from keras.layers import Conv2D, MaxPooling2D
from keras.layers import Dense, Dropout, Flatten
                  
def custom_model():
    model = Sequential()
    model.add(Conv2D(filters=16,kernel_size=(3, 3), activation='relu',input_shape=(28,28,3)))
    model.add(MaxPooling2D(pool_size=(2, 2)))
    model.add(Conv2D(filters=32,kernel_size=(3, 3), activation='relu'))
    model.add(MaxPooling2D(pool_size=(2, 2)))
    model.add(Conv2D(filters=64,kernel_size=(3, 3), activation='relu'))
    model.add(MaxPooling2D(pool_size=(2, 2)))
    model.add(Flatten())
    model.add(Dense(units=64, activation='relu'))
    model.add(Dropout(0.2))
    model.add(Dense(units=5, activation='softmax'))
    
    return model

model = custom_model()
    Solution

    28069

  5. You are given a dataset of 10 x 10 grayscale images. Your goal is to build a classifier that will classify all images into 5 classes. You decide to choose one approach from MLP and CNN:

    MLP configuration: the input is flattened into a 100-dimensional vector, followed by a fully-connected layer with 5 neurons

    CNN configuration: the input is directly given to a convolutional layer with five 10 x 10 filters

    Explain which method is better and why?

    Solution

    Suggested Answer: The two methods are essentially the same. The only difference is that the CNN configuration doesn't require flattening the input, so it's slightly more efficient in some way. But this difference can be neglected in practice.

  6. Professor Li Fei-Fei from Stanford and her team launched the ImageNet competition in 2010. And in the following years, teams from all over the world have been trying different approaches to solve the challenge. In the first 2 years, conventional image processing methods were used. And this was the common method at that time, as before the development of CNN, computer vision researchers had always been trying to find or hand engineer the features in images for multiple computer vision tasks. But in 2012, the AlexNet architecture significantly improved the performance, and CNNs became exceedingly popular after that up to today. Recall what you have learnt in the "image processing" topic, and think about where are the learnable parameters in CONV layers. Write your understanding of what CNNs can actually learn and why this would help improve the image classification performance.

    Solution

    Suggested Answer: From what you have learnt in the "image processing" topic, you should know that image convolution is powerful in finding the features of an image if we already know the right kernel to use. Using different levels of features is essential for correctly classifying the images. What Conv layers actually learn is to extract the features. Because the learnable parameters are the convolution kernel itself, it will learn to extract the useful features in the image of different levels, and use the extracted feature maps to do classification. Compared to hand engineering the features by humans, letting the model learn to extract the feature itself is less time-consuming and produces better performance results.

  10. Which of the followings is/are FALSE about the idea of the dropout?

    1. The dropout rate p is a hyperparameter to be considered when designing the model.
    2. It is designed as a way to avoid over-fitting.
    3. A dropout rate of 0.5 means half of the neuron output will be set to 0.
    4. The higher the dropout rate, the better the model performance.
    Solution

    CD

    Note that a dropout rate of 0.5 means that each neuron has a probability of 0.5 to be turned off; not exactly half of the neurons are turned off.
