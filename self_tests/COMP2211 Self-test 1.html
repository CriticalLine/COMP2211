Self-test 1:
Introduction to Artificial Intelligence

  1. Which of the following best describes AI?

    1. AI is the process of optimizing machine tasks using neural networks.
    2. AI is the science of getting machines to act in a certain way without being explicitly programmed.
    3. AI is a sentient robot.
    4. AI is the science and engineering of making intelligent machines.
    Solution
    D. AI is the science and engineering of making intelligent machines. This definition is by Alan Turing.

  2. Who is (widely regarded as) the founder of artificial intelligence?

    1. Alan Turing
    2. John McCarthy
    3. Frank Rosenblatt
    4. Desmond Tsoi
    Solution
    A. Alan Turing. John McCarthy is the first person to coin the term "Artificial intelligence"; Frank Rosenblatt is the first person to study neural networks and deep learning (perceptrons); Desmond is your instructor for this course :D

  3. Which of the following is a Python library for machine learning?

    1. Ditto
    2. Sawk
    3. Keras
    4. Vulpix
    Solution

    C. Keras. The other three are Pokemons.

  4. Which of the following fields is not a foundation of AI (for now)?

    1. Mathematics
    2. Particle physics
    3. Neurobiology
    4. Cognition science
    Solution

    B. Particle physics - but we don't know yet. Who knows about the development of AI in the future?

  5. Which of the following is not (yet) an application of AI?

    1. Gaming and Entertainment
    2. Speech Recognition
    3. Expert Systems
    4. Content mining
    Solution

    D. Content mining - it is mostly easy enough that we can hard-code a bunch of if-statements and call it a day.

  6. Which of the following Python library is (allegedly) the easiest to use?

    1. Tensorflow
    2. Pytorch
    3. Keras
    4. Scikit-Learn
    Solution

    D. Scikit-Learn

  7. Which of the following is NOT a subclass of Neural Networks? (You are encouraged to do some web searching on these algorithms as they will appear in this course :P)

    1. Multi-layer Perceptrons (MLP)
    2. Convolutional Neural Network (CNN)
    3. Minimax and Alpha-Beta Pruning
    Solution

    C. Minimax and Alpha-Beta Pruning

  8. Which of the following is a made-up event by your bad guy TA in the history of AI (Let's say 1950 - 2022)?

    1. IBM supercomputer Deep Blue beats chess champion, Garry Kasparov, in a 6-game match
    2. Ian Goodfellow comes up with General Adversarial Networks
    3. The second AI winter happened in 1987-1993
    4. The natural language processing AI, ELIZA, passes the Turing test
    Solution
    D. The natural language processing AI, ELIZA, passes the Turing test.

  9. Which of the following is not an advantage for using Python for writing AI?

    1. Python has many libraries for machine learning
    2. Python is easy to learn
    3. Python is a dynamically typed language
    4. Python is popular and has a large community
    Solution

    C. Python is a dynamically typed language. This just means Python does not store the type of variables - an integer variable might hold a string in the future. This makes Python a bit easier to use sometimes, super frustrating at other times, and as far as AI is concerned, it is mostly unrelated.
