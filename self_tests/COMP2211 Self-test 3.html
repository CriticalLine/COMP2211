Naive Bayes Classifier

  1. (Simple) True or false. The Naive Bayes classifier uses a labeled dataset.

    Solution
    True .

  2. (Simple) Which of the following are correct regarding the Naive Bayes classifier?

    1. The Naive Bayes classifier is a clustering algorithm.
    2. The Naive Bayes classifier is uses a labeled dataset.
    3. The Naive Bayes classifier assumes independence of events.

    1. (i) and (ii) only
    2. (i) and (iii) only
    3. (ii) and (iii) only
    4. (i), (ii) and (iii)

    Solution
    The answer is C. (i) is wrong because the Naive Bayes classifier is a classifying algorithm.

  4. (Medium) Consider a random bit string with length 4 (A bit string is a sequence of 0s and 1s, e.g. 0000 or 0101). What is the probability that it contains two consecutive 0s, given that the first bit is 0?

    Solution
    P( consec 0 | first bit 0 )=P( consec 0 and first bit 0 )P( first bit 0 )=5/168/16=62.5%

  5. (Advanced) On average, 1 in 1000 drivers is drunk driving. A breath analyser with an error rate of 5% is used to test if a driver is drunk driving. Police stop a driver at random and administer the breath analyser, which reports that the driver is indeed drunk driving.

    Answer the following two questions.

    1. Since the breath analyser has an error rate of 5%, a student claims the probability that the driver is drunk driving should be 95%. Explain why this intuition is wrong.
    2. Using Bayes Theorem, find the true probability that the driver is drunk driving, i.e. find P( drunk driving | reported drunk by breath analyser ).

    Solution
    1. Since the breath analyser has an error rate of 5%, in 1000 people, it would report around 50 drivers as drunk drivers, whereas amongst them only 1 driver is truly drunk. Therefore, the probability that the driver is drunk driving should only be around 150=2%. (This is just an approximation)
    2. We are given that the breath analyser reported the driver as drunk driving. Bayes Theorem is a rule which allows us to update our probabilities based on given evidence. Let D= the driver is drunk driving B= the driver is reported drunk by the breath analyser From the given information, 1 in 1000 drivers are drunk driving ⟺P(D)=0.001Error rate of 5% ⟺P(B|D)=0.95,P(B| not D)=0.05 Then, P(D|B)=P(D∩B)P(B)=P(B|D)P(D)P(B)=P(B|D)P(D)P(B|D)P(D)+P(B| not D)P( not D)=0.950.0010.950.001+0.050.999=1.87%

  6. (Medium) If two events E and F are independent, then

    1. P(E|F)=P(F|E)
    2. P(E and F)=P(E)P(F)
    3. P(E|F)=P(E)

    1. (i) and (ii) only
    2. (i) and (iii) only
    3. (ii) and (iii) only
    4. (i), (ii) and (iii)

    Solution
    The answer is C.

  7. (Medium) Let E1,⋯,En,B1,⋯,Bn be events. Denote the complement of an event E by Ec. Which of the following formulas requires the independence assumption?

    1. P(B|E1,⋯,En)=P(E1∩⋯∩En|B)P(B)P(E1∩⋯∩En|B)P(B)+P(E1∩⋯∩En|Bc)P(Bc)
    2. P(B1∩⋯∩Bn|E)=P(E|B1∩⋯∩Bn)P(B1∩⋯∩Bn)P(E)
    3. P(B1∩⋯∩Bn|E1∩⋯∩En)=P(B1∩⋯∩Bn∩E1∩⋯∩En)P(E1∩⋯∩En)

    1. (i) only
    2. (ii) only
    3. (iii) only
    4. None of the above

    Solution
    The answer is D. All of these formulas are just rearrangements of Bayes Theorem.

  8. (Medium) Which of the following is a correct description of the Naive Bayes classifier?

    1. The Naive Bayes classifier assumes each belief is independent.
    2. The Naive Bayes classifier assumes each piece of evidence contributes equally to the belief.
    3. The Naive Bayes classifier always predicts correct results.
    4. None of the above

    Solution
    The answer is B. A is wrong because it should be each evidence is independent. C is wrong because the independence assumption may not always hold, leading to incorrect results.

  10. (Simple) True or false. There are n different beliefs B1,B2,⋯,Bn. If the pieces of evidence are independent, then the following formula gives us the probability of belief B given the evidence. P(B|E1∩⋯∩En)=maxi∈{1,⋯,n}P(Bi)P(E1|Bi)P(E2|Bi)⋯P(En|Bi)

    Solution
      False. Recall that we removed the denominator in deriving this formula. It gives us which belief is the most probable but not the actual probability of it occurring. The left-hand side of the equation written above is completely wrong.

  11. (Simple) Some people prefer to use the following formula to determine the most likely belief. maxi∈{0,1,⋯,n}(ln⁡P(Bi)+ln⁡P(E1|Bi)+⋯+P(En|Bi)) What is the motivation behind such a formula?

    Solution
      Underflow prevention. Multiplying many numbers in (0,1) may cause floating-point underflow.

  12. (Simple) State one advantage and one disadvantage of the Naive Bayes classifier.

    Solution
    • Advantage: It is very easy to implement.
    • Disadvantage: The accuracy is low for a small dataset.
