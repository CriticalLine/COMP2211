Self-test 12:
Introduction to Reinforcement Learning

  1. (Simple) Which of the following are components of reinforced learning?

    1. Actions
    2. Testing data
    3. Training data

    1. (i) only
    2. (i) and (iii) only
    3. (ii) and (iii) only
    4. (i), (ii) and (iii)
    Solution
    The answer is A. The 6 components of reinforced learning are Agent, Environment, Observations, Actions, Rewards and Policy.

  2. (Medium) An animal trainer is trying to train a monkey to perform tricks in a circus. Give an example of what each of the following 6 components may refer to in training the monkey.

    1. Agent
    2. Environment
    3. Observations
    4. Actions
    5. Rewards
    6. Policy

    Solution
    1. Agent: The monkey.
    2. Environment: Circus/ training room.
    3. Observations: What the monkey observes.
    4. Actions: The tricks the monkey performs.
    5. Rewards: Food treat.
    6. Policy: Perform the trick elegantly.

  3. (Simple) Which of the following describes a Markov chain?

    1. The future is independent of the past, given the present.
    2. The present captures information about the future.
    3. Once the present is known, history may be thrown away.

    1. (i) and (ii) only
    2. (i) and (iii) only
    3. (ii) and (iii) only
    4. (i), (ii) and (iii)
    Solution
    The answer is B. The present captures information about the past, not the future.

  4. (Medium) Consider a Markov chain with processes X0,X1,X2,⋯. Match each term with each definition.

    • Definition 1: the value of Xt.
    • Definition 2: the set of values that each Xt can take.
    • Definition 3: a particular set of values for X0,X1,X2,⋯.

    1. The trajectory (also called path) of a Markov chain is...
    2. The state of a Markov chain at time t is...
    3. The state space of a Markov chain is...

    Solution
    1. The trajectory (also called path) of a Markov chain is a particular set of values for X0,X1,X2,⋯.
    2. The state of a Markov chain at time t is the value of Xt.
    3. The state space of a Markov chain is the set of values that each Xt can take.

  5. (Medium) Consider a Markov chain with processes X0,X1,X2,⋯. Which of the following is correct?

    1. X0 is a probability.
    2. X0 is a random variable.
    3. X0 is an event.
    4. X0 is a state.
    5. None of the above.
    Solution
    The answer is B. X0 is a random variable which can take on the values in the state space. This is why it makes sense to say, for example, P(X0=S0).

  6. (Medium) Consider a Markov chain with processes X0,X1,X2,⋯,XN and states S0,S1,S2,⋯,SN. Which of the following must be correct?

    1. P(X0=S0)>0
    2. P(X0=S0)+P(X1=S1)+⋯+P(XN=SN)=1
    3. P(X0=S0)+P(X1=S0)+⋯+P(XN=S0)=1
    4. P(X0=S0)+P(X0=S1)+⋯+P(X0=SN)=1
    Solution
    The answer is D.
    • Option A is wrong because the probability maybe 0.
    • Option B and C are wrong because the random variable X0 has sample space {S0,S1,S2,⋯,SN}. So summing up, the probability that X0 equals each state should be 1, i.e. option D.

  8. (Medium) Let X0,X1,X2,⋯ be the processes of a Markov chain with a state space S0,S1,S2,⋯,SN. A student makes the following claim regarding Markov chains. In general, P(Xt=S0) is independent of the values the previous processes X1,X2,⋯,Xt−1 take. Do you agree with the student's claim? Justify your answer.

    Solution
    The student's claim is false. P(Xt=S0) should be independent of the values the previous processes X0,X1,⋯,Xt−2 take, however P(Xt=S0) does not need to be independent of Xt−1, and in general, it is the case that P(Xt=S0) does, in fact, depending on the value Xt−1 takes.

    For some intuition, consider the "academic life of a professor" example used in class. The probability that the professor becomes a Full Professor at time t, expressed mathematically as P(Xt=F), depends on which state the professor was in at time t−1. If the professor was on the street at Xt−1, P(Xt=F)=0. If the professor was an associate professor at Xt−1, P(Xt=F)=0.2

  9. (Medium) A student makes the following claim regarding Markov chains. In general, each state should only occur at most once. Using the definition of a Markov chain, justify or refute the student's claim.

    Solution
    The student's claim is false. Consider if there were states A,B,C,D,⋯, and the processes of the chain looked as follows, X0=AX1=BX2=C Then, we know that X3 cannot be A because X0 was already A. However, this means that the P(X3=A) depends on what value X0 took, which violates the Markov property that X3 should only depend on what value X2 took.

    Said another way, X2=C does not capture all information of the past. It does not tell us that states A and B already occurred and hence cannot occur anymore.

  10. (Advanced) Suppose a prize box contains five 1-dollar coins, five 2-dollar coins, and five 5-dollar coins. Coins are drawn one by one from the prize box. Let Xk= the total amount drawn from the box after the kth drawM= the sequence {X0,X1,X2,⋯,X15} For example, if the first coin drawn is a 5-dollar coin, we would have X0=0,X1=5.

    Is M a Markov chain? Explain your answer.

    Solution
    M is not a Markov chain.

    To see why, consider if you are given X4=8. If you knew the previous coin draws were four 2-dollar coins, you could predict that P(X5=10)=111. However, if you didn't know, you may guess that the previous coin draws were one 5-dollar coin and three 1-dollar coins. You would then predict P(X5=10)=511.

    As you can see, the future X5 depends on X1,X2,X3. The present, X4, does not capture all information of the past.

  11. (Advanced) Suppose a prize box contains five 1-dollar coins, five 2-dollar coins, and five 5-dollar coins. Coins are drawn one by one from the prize box. Let Xk= a tuple (a,b,c), where a= the total number of 1-dollar coins drawn after the kth drawb= the total number of 2-dollar coins drawn after the kth drawc= the total number of 5-dollar coins drawn after the kth drawM= the sequence {X0,X1,X2,⋯,X15} For example, if we draw a 5-dollar coin followed by another 5-dollar coin, we would have X0=(0,0,0),X1=(0,0,1),X2=(0,0,2).

    Is M a Markov chain? Explain your answer.

    Solution
    M is a Markov chain.

    To see why, consider if you are given X1=(0,1,0). Then, you could easily determine that X2 could not possibly be something like (1,0,1) or (2,0,0). The probability of achieving certain states in X2 now depends on X1. After the second draw, the third draw now only depends on the outcome of the second draw, since all the information required to determine the probability of X3 achieving certain states is completely captured by X2.

    You could generalise the above argument to any Xk in any of the 666=216 states. Try and write a formal proof.

  12. (Simple) True or false. A transition probability matrix in a Markov chain must be a square matrix.

    Solution
    True. For a Markov chain with N states, a transition probability matrix should have a shape of (N,N).

  15. (Advanced) Consider a Markov chain with a state space {A,B,C}. The following transition probability matrix M describes the transition probabilities between two processes Xt and Xt+1, where t∈{0,1,⋯}. M=(a00a01a02a10a11a12a20a21a22)

    Prove that the transition probability matrix between X0 and X2 is given by M2.

    Hint: Use the Markov property and the fact that for an event E, P(E)=P(E|X1=A)P(X1=A)+P(E|X1=B)P(X1=B)+P(E|X1=C)P(X1=C) This formula was taught in the topic "Naive Bayes Classifier".

    Solution
    We let M2=(b00b01b02b10b11b12b20b21b22) and prove that b01=a00a01+a01a11+a02a21. The proof for the other 8 entries of M2 is exactly the same. b01=P(X2=B|X0=A)=P(X2=B|X0=A∩X1=A)P(X1=A|X0=A)+P(X2=B|X0=A∩X1=B)P(X1=B|X0=A)(Applying the hint)+P(X2=B|X0=A∩X1=C)P(X1=C|X0=A)=P(X2=B|X1=A)P(X1=A|X0=A)+P(X2=B|X1=B)P(X1=B|X0=A)(Applying the Markov property)+P(X2=B|X1=C)P(X1=C|X0=A)=a01a00+a11a01+a21a02 In the "Applying the Markov property" equality, we removed X0=A because the probability that X2=B should not depend on what state X0 took.

  16. (Medium) Consider a Markov chain with state space {A,B,C}. The transition probability matrix M describes the transitions between any processes Xt and Xt+1 of this Markov chain, where t∈{0,1,⋯}. M=(0.10.40.50.20.50.30.30.60.1) Find P(X3=C|X1=B).

    Solution
    The transition probability matrix between X1 and X3 is M2. P(X3=C|X1=B) would correspond to the entry M12, i.e. the middle row, the rightmost entry. Therefore, we do not need to compute the whole matrix M2, but only part of it. Part of M2=(−−−0.20.50.3−−−)(−−0.5−−0.3−−0.1)=(−−−−−0.28−−−) So P(X3=C|X1=B)=0.28.

  18. (Simple) State one advantage and one disadvantage of solving a Markov system algebraically.

    Solution
    • Advantage: we get an exact solution.
    • Disadvantage: if the number of states is large, solving algebraically is slow.

  20. (Medium) True or false. Consider a Markov chain with a state space {S1,S2,⋯,SN} and a discount factor γ∈(0,1). Each V1(S1),V1(S2),⋯,V1(SN) is finite. Then, limn→∞Vn(statei) converges for any statei∈state space.

    Solution
    True. It converges to (I−γM)−1, where M is the transition probability matrix (check out question 17). Intuitively, since the discount factor γ∈(0,1), the rewards decay over time, so we should expect it to converge.

  21. (Simple) Consider solving a Markov chain with state space {S1,S2,⋯,SN} by value iteration, with a threshold ε. Which of the following correctly describes the stopping condition?

    1. maxs∈{S1,S2,⋯,SN}|Vk(s)−Vk−1(s)|<ε
    2. maxsa,sb∈{S1,S2,⋯,SN}|Vk(sa)−Vk(sb)|<ε
    3. maxk∈{1,2,⋯,N}|Vk(s)−Vk−1(s)|<ε
    4. maxk∈{1,2,⋯,N}|Vk(sa)−Vk(sb)|<ε

    Solution
    The answer is A. We stop iterating when the differences between Vk and Vk−1 become small enough. We measure the difference by taking the maximum difference over each state.
